# Security Policy

## Educational Local-Only Lab

This repository is an intentionally vulnerable educational lab. It is designed
to help learners understand local web application security scenarios, structured
logging, and defensive detection logic.

The lab should only be run locally. Do not deploy the vulnerable application to
the public internet, shared infrastructure, or any environment where it could be
reached by unauthorized users.

## Responsible Use

Users must not use this project to target third-party systems, public websites,
real services, or infrastructure they do not own or have explicit authorization
to test.

All examples, logs, and detection rules should remain scoped to the local lab
application. The intended use is defensive learning, controlled testing, and
portfolio demonstration.

## Reporting Accidental Security Issues

If you find an accidental security issue in this project that is not part of an
intentional vulnerable scenario, please report it privately to the repository
maintainer.

Useful details include:

- affected file or component
- why the issue appears accidental rather than intentional
- steps to reproduce inside the local lab
- suggested defensive fix, if known

Please do not publish exploit details for accidental issues before the
maintainer has had a reasonable chance to review them.
